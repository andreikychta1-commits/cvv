Простой одностраничный сайт

Как запустить:

1. Откройте файл `index.html` в браузере (двойной клик).

Структура файлов:

- index.html — главная страница
- styles.css — стили
- script.js — простой JS для формы

Вы можете разместить эти файлы на любом статическом хостинге или открыть локально.
